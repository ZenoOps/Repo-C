Creating Customer Folders  
1. Go to the Customers folder  
2. Type the name of the customer to locate the customer folder:  
a. If no customer folder exists it will be necessary to create a main customer folder 
into which the current year needs to be added  
b. If there IS a customer folder, however, no folder for the current year, copy the 
year folder within the folder structure of the appropriate customer folder  
3. Open the year folder  
 
4. Copy the LOB folder and paste this into the relevant YEAR in the CUSTOMER 
folder you are working on.
